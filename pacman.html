Pacman

This remake of the classic Pacman game was developed in the Java and Processing.org programming languages. In this project, a player can use the four arrow keys to move the Pacman character through the maze in order to eat the pac-dots while avoiding contact with the four Ghosts, which follow a random path during the game.

If the Pacman eats one of the four large corner dots, the player briefly enters ‘Ghost Mode’ - in which Pacman can earn extra points by eating the Ghosts and sending them back to their home base. A player must successfully eat all of the dots in the maze to win the game; if the player loses all 3 lives without doing so, the game ends.
