Asteroids Game: Java and Processing.org (2017)

This project is a recreation of the classic video game, Asteroids. In my version of this game, a spaceship must avoid collision with a random number of moving and rotating asteroids. A player can earn points by shooting at and destroying the surrounding asteroids. The controls involve use of the arrow keys to move and rotate the ship, and the ‘h’ key to use ’hyperspace mode’ to move the spaceship to a random position. Asteroids was developed with the programming languages Java and Processing.org.
